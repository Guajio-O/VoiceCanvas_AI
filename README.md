# 🎨 VoicePaint - 语音绘图工具

一款纯语音控制的智能绘图工具，通过语音指令即可完成绘图创作。

[English](README_EN.md) | 中文

## ✨ 功能特点

### 🗣️ 语音控制
- **纯语音操作**：无需鼠标或键盘，通过语音指令完成绘图
- **中文支持**：原生中文语音识别，指令理解更自然
- **智能纠错**：指令理解准确率高，支持多种表达方式

### 🎯 双模式绘图
- **本地模式**：使用 HTML5 Canvas 绘制基础图形，响应迅速
- **AI模式**：调用 AI 绘图 API 生成高质量图片

### 🎨 本地绘图支持图形
| 类别 | 图形 |
|------|------|
| 基础 | 圆、矩形、三角形、椭圆、线段 |
| 高级 | 五角星、心形、十字、菱形、五边形 |
| 创意 | 太阳、月亮、房子、树、彩虹、钻石、箭头、波浪 |

### 🤖 AI 绘图支持
- 支持 **Hugging Face** API
- 支持 **OpenAI DALL·E** API
- 支持 **Stable Diffusion** 本地/远程 API
- 支持**任何兼容的绘图 API**
- API 配置完全开放，用户可自行填入

### 🔧 指令能力
- **画布控制**：清空、撤销
- **颜色切换**：12种颜色（红、橙、黄、绿、青、蓝、紫、黑、白、棕、粉、灰）
- **线条调整**：粗细调节（粗一点、细一点）
- **填充样式**：实心、空心
- **尺寸调节**：大、小、更大、更小
- **表情变体**：笑脸太阳、哭脸月亮
- **位置关系**：支持左/右/上/下/内部等方位（如"猫在车左边"、"鱼在水里"）
- **组合指令**：支持从长句中提取多个关键词同时执行

## 🚀 快速开始

### 方法一：直接打开（本地模式）
1. 双击 `index.html` 打开应用
2. 选择"本地绘图模式"
3. 点击"开始监听"，开始语音绘图

### 方法二：本地服务器（AI模式，推荐）
1. 确保已安装 [Node.js](https://nodejs.org/)
2. 进入项目目录
3. 运行服务器：
   ```bash
   node server.js
   ```
4. 浏览器访问：`http://localhost:8000`
5. 配置 AI API（API地址、API Key）
6. 选择"AI绘图模式"，开始创作

## 📖 语音指令示例

### 本地模式
| 指令 | 效果 |
|------|------|
| "画一个圆" | 绘制圆形 |
| "画矩形" | 绘制矩形 |
| "用红色" | 切换为红色 |
| "粗一点" | 线条加粗 |
| "清空" | 清空画布 |
| "撤销" | 撤销上一步 |
| "画一个笑脸太阳" | 绘制带笑脸的太阳 |

### AI模式
| 指令 | 效果 |
|------|------|
| "画一只可爱的猫" | AI生成猫咪图片 |
| "画美丽的风景" | AI生成风景图片 |
| "画一个科幻城市" | AI生成科幻城市场景 |

### 组合指令
| 指令 | 提取的关键词 |
|------|-------------|
| "画一个带笑脸的黄色太阳" | 黄色 + 笑脸太阳 |
| "画实心的大圆" | 实心 + 大 + 圆 |
| "画蓝色的小星星" | 蓝色 + 小 + 星星 |

## ⚙️ 配置说明

### API 配置
在设置面板中可以配置以下选项：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| API 地址 | 绘图API的基础URL | `https://api-inference.huggingface.co` |
| API Key | 访问API的密钥（不会保存到代码中） | `hf_xxxxx` |
| 模型名称 | AI模型标识符 | `stabilityai/stable-diffusion-xl-base-1.0` |

**提示**：API地址、Key和模型名称都需要填写才能使用AI绘图功能。

### 本地模式配置
无需配置，直接可用

## 🛠️ 技术栈

- **前端**：HTML5 + CSS3 + JavaScript（原生实现，无框架依赖）
- **语音识别**：Web Speech API
- **语音合成**：Web Speech Synthesis API
- **绘图**：HTML5 Canvas
- **服务器**：Node.js（仅用于AI模式跨域代理）

## ⚠️ 注意事项

1. **语音识别需要麦克风权限**，请在浏览器中允许使用麦克风
2. **AI模式需要网络连接**，且需要有效的 API Key
3. **推荐使用 Chrome 或 Edge 浏览器**，语音识别支持更好
4. **本地模式可离线使用**，无需服务器
