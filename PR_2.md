# PR 描述：复杂位置关系处理与增量绘制功能

## 标题
feat(position): 实现复杂位置关系处理与增量绘制功能

---

## 功能描述

本 PR 为语音绘图工具增强了位置关系处理能力：

1. **复杂方位检测**：支持"左/右/上/下/前/后"等方位词的精确检测
2. **增量绘制**：基于已有图形位置添加新元素，无需重复绘制参考对象
3. **位置关系规则**：支持"在内部"、"在上方"、"在旁边"等语义关系
4. **方位修复**：修复"狗在右边却绘制在左边"的问题

---

## 功能详情

### 1. 方位词检测增强

新增 `detectDirectionWithContext()` 函数，支持更精确的方位检测：

| 指令示例 | 检测结果 |
|---------|---------|
| "猫在车左边" | left |
| "狗在车右边" | right |
| "鸟在房子上面" | top |
| "鱼在水里" | inside |

### 2. 位置关系规则

扩展 `POSITION_RULES`，支持多种位置关系：

```javascript
const POSITION_RULES = {
    inside: {  // 在内部
        '鱼': ['河', '湖', '海', '水'],
        '海豚': ['河', '湖', '海', '水'],
    },
    onTop: {   // 在上方（支持方位）
        '猫': ['汽车', '飞机', '船', '火车'],
        '狗': ['汽车', '飞机', '船', '火车'],
        '鸟': ['房子', '山'],
    },
    beside: {  // 在旁边
        '猫': ['汽车', '房子', '树'],
        '狗': ['汽车', '房子', '树'],
    }
};
```

### 3. 增量绘制支持

支持基于画布已有图形进行增量绘制：

```
场景：先绘制"猫在车左边"，再补充"狗在车右边"
结果：汽车已存在，直接在其右边绘制狗
```

### 4. 复杂指令处理

支持多位置关系同时处理：

| 指令 | 期望结果 |
|------|---------|
| "猫在车左边，狗在车右边" | 车居中，猫在左，狗在右 |
| "猫在汽车上，鱼在水里" | 两组独立位置关系 |

### 关键代码

```javascript
// 方位检测 - 先检查参考形状后面的文本
const contextAfterRef = textLower.substring(refPos, refPos + 20);
for (const [chinese, direction] of Object.entries(DIRECTION_WORDS)) {
    if (contextAfterRef.includes(chinese)) {
        return direction;
    }
}

// 位置绘制 - 根据方位调整坐标
if (pair.direction === 'left') {
    addLocalShapeAt(pair.top, bottomX - 100, bottomY - 40);
} else if (pair.direction === 'right') {
    addLocalShapeAt(pair.top, bottomX + 100, bottomY - 40);
}
```

### 技术要点

- **上下文感知**：方位词通常位于参考形状之后
- **增量绘制**：保留已有图形，避免重复绘制
- **精确坐标计算**：根据方位词调整绘制位置
- **多关系支持**：同时处理多组位置关系

## 测试方式

### 步骤 1：测试基本方位

1. 切换到「本地绘图模式」
2. 说出指令："画一只猫在车左边"
3. 确认：汽车在中间，猫在汽车左侧

### 步骤 2：测试增量绘制

1. 继续上一步场景
2. 说出指令："画一只狗在车右边"
3. 确认：汽车保留，狗在汽车右侧

### 步骤 3：测试复杂指令

1. 清空画布
2. 说出指令："猫在车左边，狗在车右边"
3. 确认：车居中，猫在左，狗在右

### 步骤 4：测试内部关系

1. 说出指令："画一条鱼在水里"
2. 确认：鱼绘制在水/椭圆内部

## 修复的问题

| 问题 | 原因 | 修复方案 |
|------|------|---------|
| 狗在右边却绘制在左边 | 方位词检测范围错误 | 扩展搜索范围到参考形状之后 |
| 增量绘制时参考对象重复绘制 | 未检查已有图形 | 先查找画布上的参考形状 |
| 方位词"right"被识别为"left" | 英文方位词检测顺序问题 | 按上下文位置精确匹配 |
